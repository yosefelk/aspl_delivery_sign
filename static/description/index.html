Odoo 13 Delivery Signature

User can validate picking with signature.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • User can validate picking after add signature.

Available for Enterprise

How to use

  • Configuration for enable signature.
  • As per below image we can see configuration of signature.
  • When we enable 'Require Signature' then we must have to add signature for validate Picking.

  • After configure signature when user click on 'Validate' at that time generate warning like below image.

  • As per below image we can see 'Signature' button.

  • 'Signature' button click open new pop up for add signature.

  • After add signature we can validate this picking.

  • If signature is added on picking then it will also print on Picking Operations reports.
